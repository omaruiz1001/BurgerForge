# 🍔 Burger Forge - Landing Page

Proyecto de una hamburguesería artesanal con sistema de carrito de compras integrado con WhatsApp.

## ✨ Características
- 📱 **Diseño Responsivo:** Optimizado para móviles y tablets.
- 🛒 **Carrito Dinámico:** Gestión de pedidos en tiempo real.
- 🚀 **SEO Avanzado:** Implementación de JSON-LD y Meta-tags.
- 💬 **WhatsApp Integration:** Envío automático del detalle del pedido.

## 🛠️ Tecnologías
- HTML5 / CSS3 / JavaScript (Vanilla)
- Google Fonts & Unsplash API
- Despliegue en Netlify
